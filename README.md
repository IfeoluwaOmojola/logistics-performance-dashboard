# logistics-performance-dashboard
Power BI dashboard analyzing revenue, fleet utilization, and incident trends (2022–2024)
📊 Logistics Operations Dashboard
🔍 Overview

This project analyzes logistics performance, revenue, and incident trends from 2022 to 2024 using Power BI.

❓ Business Questions
1. How is revenue and profit performing over time?
2. What is the fleet utilization rate?
3. What factors affect operational efficiency?
4. What trends exist in incident reports?

🛠 Tools Used
1. Power BI
2. Data Modeling
3. DAX
4. Data Cleaning

📈 Key Insights
1. Generated ₦262.53M revenue and ₦161.20M profit
2. 76.67% fleet utilization, indicating strong operational efficiency
3. Incident rates peaked mid-year but declined later
4. DOT violations were the most common incident type

💡 Recommendations
1. Improve compliance processes to reduce DOT violations
2. Monitor mid-year operations to reduce incident spikes
3. Optimize fleet allocation to maintain high utilization
